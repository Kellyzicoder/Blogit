ABOUT US

A New Zealand online content management system which enables multi-user blogs with time-stamped entries. Google hosts the blogs, which can be accessed by a user anytime, anywhere by using any browswer. Blogit also enabled users to publish blogs and save as draft. Blogit has a wide international user base and is available in more than 60 languages, despite its decline in popularity in the United States.
